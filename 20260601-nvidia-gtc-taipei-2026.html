RTX Spark — 重新发明 PC

"过去四十年，计算机是为人造的。从今天起，它要为 Agent 造。"

RTX Spark (N1X) 是英伟达与微软联手打造的全球首款 Agent PC 芯片。6144 CUDA 核心，128GB LPDDR5X 统一内存，1 PFLOP FP4 AI 算力，20 核 Grace CPU（联发科合作），台积电 3nm，700 亿晶体管。14mm 笔记本即能本地跑 70B–120B 参数模型，支持 LoRA 微调、RAG、Agent 自迭代。

关键护城河：第一次在同一颗芯片上实现统一内存 + 完整 CUDA 生态——Mac 有内存没 CUDA，传统 N 卡有 CUDA 但被显存卡死。

Vera Rubin：下一代数据中心全面量产

72 块 Rubin GPU + 36 颗 Vera CPU 组成 NVLink 72 超级计算单元。供应链规模 2× Grace Blackwell，机架组装从 2 小时缩至 5 分钟，无电缆/无软管/无风扇设计。超过 100 万 MGX 组件来自台湾 25 个工厂，500+ 合作伙伴。

Cosmos 3：全开源全模态物理 AI 模型

基于 Mixture-of-Transformers 架构，原生理解和生成文本、图像、视频、环境声音、动作数据五合一。世界生成排名第一，训练从数月缩至数天，完全开源。成立 Cosmos 联盟（思灵机器人、Runway、Skild AI 等）。

Agent 基础设施四层架构

模型层 Nemotron 3 Ultra + Cosmos 3
框架层 Claude Code / Codex / OpenClaw 直跑
工具层 CUDA-X Skills
运行时层 OpenShell（Apache 2.0 开源安全运行时）

Isaac GR00T：全球首个开源人形机器人

1.8 米，68 公斤，75 DOF，宇树科技 2026 年底生产。宇树 H2 Plus 本体 + Sharpa Wave 五指灵巧手 + Jetson AGX Thor T5000。Ai2、ETH Zurich、Stanford Robotics 已确认采用。

DSX：从卖芯片到卖整座 AI 工厂

NVIDIA DSX 提供 AI 工厂完整施工蓝图——数字孪生模拟 + DSX OS 接管运营。老黄预言到本世纪末 100 吉瓦 AI 工厂陆续上线。
