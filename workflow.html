1 工作流概览

你发一个链接（微信文章、网页、研究报告），我自动完成剩下的所有事情：

  • 抓取网页内容，解析正文
  • AI 深度分析，提取核心观点、关键数据、精彩引用
  • 用知识页面模板排版，生成精美的 HTML
  • 推送至 GitHub Pages 公开站点
  • 把可分享的网页链接发给你

整个过程 你只需要做一件事：把链接丢过来。

3 如何使用

在任何对话中，直接发给我一个链接即可。支持的类型：

  • 微信公众号文章 — 自动抓取正文，忽略广告和无关元素
  • 研究报告 / 博客 — 深度分析与知识提炼
  • 新闻 / 资讯 — 提取关键信息与数据
  • 任何公开网页 — 只要可以访问

生成后的页面会自动出现在 AI Insights 站点首页，所有人都能通过网页访问。
